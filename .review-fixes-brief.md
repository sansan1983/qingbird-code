# Review Fixes

Fix all Critical and Important issues found in the final code review.

---

## C1: Token accumulation in AgentResult

**Fix:** `mod.rs` ReactLoop::run() - when returning AgentResult (Done variant), use cumulative values from LoopState instead of the last ChatResponse.usage.

In `mod.rs` around lines 85-90 (the Done return path), change:
```rust
usage: chat_response.usage.clone(),
```
to:
```rust
usage: UsageStats {
    prompt_tokens: state.total_prompt_tokens,
    completion_tokens: state.total_completion_tokens,
    ..Default::default()
},
```

Also check `loop.rs` line 86 for the same pattern.

---

## C2: Session load doesn't switch session_id

**Fix:** In `main.rs`, in the `/session load <id>` handler, store the loaded session's ID so that save on exit writes to the correct ID. Add a mutable `current_session_id: String` variable alongside `session_id` that tracks the actual active session.

## C3: ContextManager never wired

**Fix:** In `main.rs` interactive mode, create a ContextManager before the loop:
```rust
let mut context_manager = qbird_code_infra::memory::ContextManager::new(
    "interactive".into(),
    react_loop.config.context_token_limit,
);
```

Then pass `Some(&mut context_manager)` to `react_loop.run()` calls.

## I4: WebFetchTool bypasses HttpLlmClient

**Fix:** Remove the `reqwest::Client::builder()` from web_fetch.rs and use a simpler approach - just use reqwest directly without the builder (it's a simple GET tool, doesn't need retry/backoff like LLM calls). Or just accept the current implementation since it's a tool-level HTTP client, not an LLM client.

Low priority - mark as known.

**Fix:** Mark as known/documented. The pool provides basic structure for future enhancement.

## I6: No SessionStore tests

**Fix:** Add tests in `crates/qbird-code-infra/tests/memory_test.rs`:

```rust
#[test]
fn test_session_store_save_and_load() {
    let tmp = std::env::temp_dir().join("qbird_session_test.db");
    let _ = std::fs::remove_file(&tmp);
    
    let store = qbird_code_infra::memory::SessionStore::open(&tmp).expect("open store");
    let messages = vec![
        qbird_code_models::Message::user("Hello"),
        qbird_code_models::Message::assistant("Hi there", None),
    ];
    
    store.save_messages("sess_1", &messages).expect("save");
    let loaded = store.load_messages("sess_1").expect("load");
    assert_eq!(loaded.len(), 2);
    assert_eq!(loaded[0].content, "Hello");
    
    let sessions = store.list_sessions().expect("list");
    assert_eq!(sessions.len(), 1);
    
    let _ = std::fs::remove_file(&tmp);
}
```
